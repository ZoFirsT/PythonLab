## Passenger by Embarked Ports and Class

จงเขียนโปรแกรมเพื่อแสดงข้อมูลผู้โดยสารที่ขึ้นเรือจากท่าเรือต่าง ๆ จากข้อมูลผู้โดยสารที่ขึ้นเรือโดยสาร `data.csv` 

โปรแกรมจะเลือกผู้โดยสารที่จากแต่ละท่าเรือก่อนโดยใช้คอลัมน์ `'Embarked'` หลังจากนั้นโปรแกรมจะเรียงลำดับผู้โดยสารตามประเภทชั้นโดยสาร (`'Pclass'`) จากน้อยไปมาก ตามด้วยราคาตั๋ว (`'Fare'`) จากมากไปน้อย 

หลังจากที่เรียงลำดับแล้วให้แสดงผลผู้โดยสารเฉพาะคอลัมน์ `['Pclass','Fare','Embarked']` จาก 5 คนแรก และ 5 คนสุดท้าย

**หมายเหตุ** : หากไม่ได้เขียนโปรแกรมโดยใช้ `pandas` เพื่อนับจำนวน จะไม่ได้คะแนน

**Hint**
* ชื่อท่าเรือที่ผู้โดยสารขึ้นเรือสามารถดูได้จากคอลัมน์ `'Embarked'` โดยจะมีทั้งหมด 4 แบบ คือ: `'S','C','Q','UNK'`
* ประเภทของชั้นของผู้โดยสารสามารถดูได้จากคอลัมน์ `'Pclass'`
* ราคาตั๋วของผู้โดยสารสามารถดูได้จากคอลัมน์ `'Fare'`
* สามารถใช้ `sort_values()` ในการเรียงลำดับผู้โดยสารได้ โดยสามารถระบุให้เรียงจาก *น้อยไปมาก* หรือ *มากไปน้อย* ด้วยการตั้งค่า `ascending` เป็น `True` หรือ `False`
* สามารถใช้ `head()` และ `tail()` ในเลือกข้อมูลผู้โดยสาร 5 คนแรก และ 5 คนสุดท้าย

โปรแกรมจะแสดงข้อมูลผู้โดยสารที่ขึ้นเรือจากแต่ละท่าเรือด้วยรูปแบบด้านล่าง
```
Embarked: <ชื่อท่าเรือ>
<ข้อมูลผู้โดยสารหลังจากเรียงลำดับแล้ว 5 คนแรก>
<ข้อมูลผู้โดยสารหลังจากเรียงลำดับแล้ว 5 คนสุดท้าย>
```
